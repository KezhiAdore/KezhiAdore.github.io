电脑软件配置单

发表于 2023-11-06 更新于 2023-11-10 分类于 Tips Valine：
本文字数： 972 阅读时长 ≈ 1 分钟

我过去经常帮身边的朋友指导电脑硬件配置。然而，对于个人电脑来说，如果胡乱装软件，或者用了不合适的软件，会让电脑的使用效率事倍功半。同时，在我个人拿到一台新电脑时，也需要根据我的习惯初始化配置很多软件到新电脑上。由此，便有了写一份软件配置清单的想法，一来可以给大家做一些软件推荐，二来我自己也留一份作为参考。

工具类

  • 火绒安全软件：台式机上可装，拦截弹窗很方便，火绒安全 (huorong.cn)

  • 达芬奇：好用的视频剪辑调色软件，DaVinci Resolve 18 | Blackmagic Design

  • OBS Studio：究极好用的开源录屏软件，Open Broadcaster Software | OBS (obsproject.com)

  • 网易邮箱大师：邮箱管理软件，网易邮箱大师-高效强大的全平台邮箱客户端 (163.com)

  • BandZip：压缩解压工具，现在有广告了QAQ，Windows 版本的 Bandizip · 业内领先的压缩文件解决方案 (bandisoft.com)

  • Geek Uninstaller，Geek Uninstaller - Download

  • PotPlayer：免费开源强大的视频播放器，Global Potplayer (daum.net)

  • SeaDrive：seafile网盘挂载工具，下载 - Seafile

学习类

  • Zotero：究极好用的文献管理工具，Zotero | Your personal research assistant，插件移步Zotero插件推荐及下载 | Kezhi's Blog

  • Typora：究极好用的Markdown编辑工具，Typora — a markdown editor, markdown reader.

  • PicGo：与Typora搭配使用，管理图床图片，PicGo (molunerfinn.com)

  • Pandoc：好用的Markdown渲染工具，与Typora配合使用，Pandoc - index

  • 网易有道词典：并不好用的翻译工具，奈何找不到更好的，网易有道 (youdao.com)

开发类

  • Visual Studio Code：最好的代码编辑器，Visual Studio Code - Code Editing. Redefined

  • Git：版本管理工具，Git (git-scm.com)

  • Anaconda3：好用的Python环境管理工具，Free Download | Anaconda
